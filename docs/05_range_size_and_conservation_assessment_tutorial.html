05_ range_size_and_conservation_assessment

Library setup

library(speciesgeocodeR)
library(ConR)
library(tidyverse)

1. Approximate species ranges

dat <- read_csv("inst/occurrence_records_clean.csv") %>% dplyr::select(species, 
    decimallongitude = decimalLongitude, decimallatitude = decimalLatitude)

# Based on EOO
rs <- CalcRangeSize(dat)

# Limited to biomes with records Load Olson et al 2001 biomes
biom <- WWFload(x = "inst")
names(biom)

rs_biome <- CalcRangeSize(dat, biome = biom)

range <- data.frame(rs, rs_biome)

2. Automated conservation assessment

You can use the ConR package for a preliminary conservation assessment orientated on the IUCN Red list Criterion B. THis is based on the EOO we have encountered above and additionally the Area of OCcupancy and the number of subpopulations.

# Format input data
dat <- dat %>% dplyr::select(ddlat = decimallatitude, ddlon = decimallongitude, 
    tax = species)


# Preliminary assessment
ev <- IUCN.eval(dat)
